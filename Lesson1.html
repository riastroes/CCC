When you start in openprocessing, check the settings.
You need to start in P5 mode. And check the option Show the console. It will give you error information.

Every P5 sketch will start with two function: 

          function setup(){
             instruction 1;
             instruction 2;
             instruction 3;
             etc.
          }

          function draw(){
             instruction 1;
             instruction 2;
             instruction 3;
             etc.
             start again with instruction 1;
          }

In the setup function, between the curly brackets you can write instruction for the computer. These instructions will be executed from top to bottom. Every instruction has to end with a semicolumn. When the webpage is loaded, P5 will call the setup function to execute. After the computer has executed the setup function it will call the draw function. The draw function will be executed 60 times a second . In the draw function you write your drawing instructions for computer.

In the setup function you have to prepare your canvas. In the example above is a canvas created: 

        createCavas(windowWidth, windowHeight);

createCanvas is a function that needs two params in wich you specify the width and the height of the canvas.
windowWidth is a variable of P5 that stores the width of your screen in pixels.
windowHeight is a variable of P5 that stores the height of your screen in pixels.

You can make a smaller or bigger canvas by changing the params

        createCanvas(500, 400);

Now the width is 500 pixels and the height is 400 pixels.
Can you see the difference between a function and a variable?
A functions always has brackets (), even if the function does not need any params. Notice that functionsnames and variablesnames both are written in 'camelcase':

  • the first word in the name is always starting with a small character.
  • the next words in the name are always starting with a capital.
  • the words are written together, without spacing or underscores.
