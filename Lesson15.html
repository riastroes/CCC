Lesson 15: Iteration


One of the strong points of a computer is to do some iteration. So of course there is a function to support this structure. With the for function you can control the iteration.

for(begin, test, step)

  • You will learn

  • iteration
  • local variable
  • ++
  • --
  • debug
  • println
  • console
  for(var i = 0; i < 4; i++){
    //do something
  }

The for function needs 3 special params, they are separated by a semicolon; the first is begin: 

    var i = 0;

Here we declare and initialize a local variable, this variable is used in this function to count the loops of an iteration. The second param is a test, to check if the end of the interation is not reached. The function for will be executed as long as the test is true.

    i < 4

In this case the iteration will go on as long as i is smaller then 4. The third param is the step-size of the counter.

    i++  ( is the same as i += 1)

After every loop there will be one added to the counter i. When you construct a for function, you should check if it is possible for i to reach the end. In this case is it possible for i to reach 4. Otherwise your computer will crash! 

function setup() {
  createCanvas(400,300);
  background(100);
}

function draw() {
  stroke(200,255,255);
  fill(255,255,200);
  if(frameCount == 1){
    for(var i = 0; i < 4; i++){
      ellipse(random(width),random(height), 30,30);
    }
  }
}

In the for function you can use this counter (i), but don't change this counter it self. (tip) i is a local variable, it is declared in a loop of the draw function, so it will only be known during this loop. The param is given to the for function from the draw function. You can check the value of a variable with the println function. The println function will write the stored value in the console.

  background(100);
  x = 0;

}

function draw() {
  stroke(200,255,255);
  fill(255,255,200);
  for(var i = 0; i < 4; i++){
    x = i * 50;        //i = 0, 1, 2, 3
    ellipse(x, y, 30,30);
    rect(x-15,y+15,30,60);
  }
  println(i);
}

var x, y;
function setup() {
  createCanvas(400,300);
  background(100);
  x = 0;
  y = 0;
}

function draw() {
  for(var x = 0; x < width; x += 50){
      if(x === 100){
         fill(255,200,200);
         ellipse(x,100, 50,50);
      }
      noStroke();
      fill(255,255,200);
      rect(x,y,20, height);
      println(x);
  }
}
