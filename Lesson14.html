Lesson 14: Fonts


In lesson 12 you started using text, and you got a default font, but you can use several fonts. There are web safe fonts. Fonts that look the same on different displays.
On the site of w3schools there is a list of web safe fonts. Your browser has several fonts available, but every browser is different. If a font is not loading, you need an external font. You can load many free fonts from the website

The text function displays a text on the canvas.
Before you can display any text you have to prepare some tools. 

    fill(0)
    textSize(size);       //fill in the size of the text.
    text("sometext", 30, 100);
These functions will display "sometext" in black in a default (standard) font, with a fontsize of 30 pixels. The smalles size for a character is 6 pixels.

Allways when you want to use letters, words or phrases, you have to place them between quotes. 

"a" or 'a'
"word" or 'word'
"This is a phrase." or 'This is a phrase.'
In javascript this is called a string. For the special characters and signs that are not on you keyboard or are reserved by html, you have to use HTML entities. You can find a reference of HTML entities at www.w3schools.com. The next example will only play/.................................. 

var fontDancing;
function preload() {
   fontDancing = loadFont("DancingScript-Regular.ttf");
}
function setup() {
  createCanvas(windowWidth, windowHeight);
  background(100);
  textFont(fontDancing);
  textSize(70);
  textAlign(CENTER);

  fill(255);
}

function draw() {
  text("The name of this font is DancingScript-Regular.",width/2, height/2);
}
