If ...

If you want to be in control use if. 

    If(condition){
      do this
    }

Until now the code in your setup- and draw-function was executed from top to bottom. And every line was executed. Now you will get more control over the execution of the commands with this conditional statement.

Examples of conditions with different operators:

  • if (x > 100) // if x is greater then 100.
  • if (y < 0 ) // if y is less then 100.
  • if (x = 10) // WRONG!! this is an assignment. You store 10 into a variable x
  • if (x == 10) // OK if x is equal to 10 or eqaul to "10".
  • if (x == 0) // OK if x is equal to 0 or 'undefined' or "0".
  • if (x === 0) // if x is equal to 0.

Pay attention to the difference between == and ===. In the computer 0 has more meanings then zero. It is also 'undefined', a variable has no value yet. And in javascript we don't always write our numbers as digits, but sometimes as strings. Your code will be more solid if you use the === operator.
The most solid solution is to use the three equalsigns.

If I am a big tree, I will have apples. 

    var x,y;
    var size;
    function setup(){
      createCanvas(400, 500);
      background(200);
      x  = width/2;
      y =  height;
      size = 0;
    }
    function draw(){
      if(y > 200){
        //grow
        stroke(0);
        strokeWeight(30);
        point(width/2,y);
        y -= 1;
      }
      if(y === 200){
        //leafs
        noStroke();
        fill(0,100,0,10);
        ellipse(x, y, size, size);
        size += 1;
      }
      if(size > 200){
        //apples
        noStroke();
        fill(255,255,0);
        ellipse(random(x - (size/2), x + (size/2)),
                random(y - (size/2), y + (size/2)), 30,30);
      }
      if(size === width){
        noLoop();
      }

    }
