The text function displays a text on the canvas.
Before you can display any text you have to prepare some tools. 

    fill(0)
    textSize(size);       //fill in the size of the text.
    text("sometext", 30, 100);
These functions will display "sometext" in black in a default (standard) font, with a fontsize of 30 pixels. The smalles size for a character is 6 pixels.

Allways when you want to use letters, words or phrases, you have to place them between quotes. 

"a" or 'a'
"word" or 'word'
"This is a phrase." or 'This is a phrase.'
In javascript this is called a string. For the special characters and signs that are not on you keyboard or are reserved by html, you have to use HTML entities. You can find a reference of HTML entities at www.w3schools.com. 

var sometext;

function setup() {
  createCanvas(400,400);
  background(0);
  sometext ="HELLO";
}

function draw() {

  fill(random(255));
  textSize(30);
  text(sometext, random(width-100), random(30,height-25));

  fill(random(255),0,0);
  textSize(20);
  text(sometext, random(width-65), random(20,height-5));
}

p5 will try to wrap your text if you define a rectangle in which it has to display the text. You need two more params.
The x, y, rectwidth, rectheight define a rectangle.
ps: If your box is to small it will not display your text. 

fill(0)
textSize(size);       //fill in the size of the text.
text("this is a wrapped block of text on a canvas", x, y, rectwidth, rectheight);
          


function setup() {
  createCanvas(400,400);
  background(0);
}

function draw() {
  fill(255);
  textSize(20);       //fill in the size of the text.
  text("this is a wrapped block of text on a canvas", 50,20,30, 350);

  //show rectangle if mouse is pressed
  if(mouseIsPressed){
    noFill();
    stroke(255);
    rect(50,20,30, 350);
  }
}

If you wrap your text in a block, you can choise to align the text.

    textAlign(CENTER);
For the textAlign function you can use the values:
LEFT (default), CENTER and RIGHT 


function setup() {
  createCanvas(400,400);
  background(0);
}

function draw() {
  fill(255)
  textSize(30);       //fill in the size of the text.
  textAlign(CENTER);
  text("this text is placed in the center of the defined rectangle.", 50,50, 200,300);

  //show rectangle if mouse is pressed
  if(mouseIsPressed){
    noFill();
    stroke(255);
    rect(50,50, 200,300);
  }
}
