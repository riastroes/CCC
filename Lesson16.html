Lesson 16: Images


If you want to use an image in a sketch, you have to upload this image to the web. And as well as with fonts you have to do this in the preload function, otherwise your image object will be empty, if the setup function or draw function has to use it. Be aware of the size of your image, if your image is too big, your viewers will be annoyed. There are a lot resize tools on the internet available like picresize.com
Online Image editors can help you to make the background of your image transparent. On windows I use paint.net it is a powerfull Free Image Editor. If you make the background transparent your image can nicely blend into your sketch and width the resize function you can ajust the size of your image to your canvas.
An endless source of images is Google Images.

Images

The function image has three params. The first param is the image object, the second and the third is the position on the canvas, where to put the image. Just like a rectangle the image is set on position x, y with his left-top-corner. 

            image( img, x, y);

Before you set the image on your canvas, you can ajust the size of the image. A image is an object and an object can have functions or methods and properties. In this case we call the resize function of the image. 

              img.resize(50,80);
As you can see the function is added to your variable img with a dot in between. the first param is the new width of the image object and the second param is the new height of the param. If you enlarge an image you will lose graphic quality, but is save to make an image smaller. But it is not a good idea to upload very big images, your viewers will have to wait until all the images are loaded, and they don't like that.

An image and a rectangle are placed on the canvas relative to his left-upper-corner. This is not always convenient and you can change this by adjusting the imageMode and the rectMode. Default the imageMode and rectMode are in these modes: 

              imageMode(CORNER);
              rectMode(CORNER);
These are the other options: 
              imageMode(CENTER);  // this position of x and y are anchored at the center of the image
              rectMode(CENTER);   // this position of x and y are anchored at the center of the rect

              imageMode(CORNERS);
the first two params of the image function are the upper-left-corner and the second and fourth param are now the bottom-right-corner. imageMode(CORNERS); Rectangle do have another modes: 
            rectMode(RADIUS);
Now the third and fourth params are half the width and half the height of the rectangle. Do mind the capitals, because javascript is case-sensative. So it will not understand center, corner, etc. 

  var img;
  function preload(){
    img = loadImage("ria.png");
  }
  function setup() {
    createCanvas(400,400);
    background(200);
    img.resize(400,400);
    imageMode(CENTER);
    rectMode(CENTER);
  }

  function draw() {

    image(img,width/2,height/2);
    stroke(90, 0, 23);
    strokeWeight(10);
    fill(48, 70, 102)
    ellipse(width/2, 100,400,100);

    fill(140, 70, 106);
    rect(width/2,70,200,70);
    noStroke();
    ellipse(width/2, 100,190,50);
    strokeWeight(10);
    stroke(90, 0, 23);
    fill(48, 70, 102);
    ellipse(width/2, 35,200,50);

    strokeWeight(8);
    stroke(90, 0, 23)
    fill(48, 70, 102);
    ellipse((width/2) +  80, 80, 100,100);
    strokeWeight(4);
    stroke(140, 70, 106);
    fill(116, 4, 38);
    ellipse((width/2) +  110, 100, 100,100);
    ellipse((width/2) +  90, 110, 50,50);
    ellipse((width/2) +  80, 150, 30,30);
    ellipse((width/2) +  100, 170, 30,30);
  }

  var teapot;
  var donkey;
  function preload(){
    teapot = loadImage("teapot.png");
    donkey = loadImage("donkey.png");
  }
  function setup() {
    createCanvas(400,400);
    background(200);
    imageMode(CENTER);
    rectMode(CENTER);
  }

  function draw() {
    image(teapot,width/2,height/2);
    image(donkey,width/2 + 75,height/2 -95);
    textSize(20);
    textAlign(CENTER);
    fill(0);
    text("This tea is hot!", width/2, height/2 + 100);
    noLoop();
  }
