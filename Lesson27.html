Lesson 27: EXPORT PDF


Why should you want to export to pdf? PDF is scalable, it is vector-based and not bitmap(dots) based.

If you use a svg renderer for the creation of your canvas your drawing is a vector image and you can export it as svg or as pdf.

  • You will learn

  • include p5.svg library
  • PDF
  • SVG
  • createPDF
  • beginRecord
  • save

Export PDF


    


      

      var pdf;

      function setup() {
          // If we use SVG Renderer, then the PDF generated will be vector
          // Note that to use SVG Renderer, you must include p5.svg library
          createCanvas(600, 200, SVG);
          pdf = createPDF();
          pdf.beginRecord();
      }

      function draw() {
          background(255);
          fill('#ED225D');
          textSize(100);
          textAlign(CENTER);
          text(frameCount, width * 0.5, height * 0.5);
          if (frameCount == 10) {
              noLoop();
              pdf.save();
          }
      }
