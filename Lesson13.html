Lesson 13: Fade out


The fading out effect is used to let thing dissapear slowly in the background. After you present a title, it is a nice effect to let it fade out. But not only text can be faded out. In this lesson you will see this fading out effect in several examples.

  • You will learn about

  • the fade out effect

Fade out effect

In the next example you will see the difference when cicles are overwritten by a black background or when circles are fading out. The last effect is a lot softer. You can create a fade out effect by drawing a transparent background in each frame. 

function setup() {
  createCanvas(400,400);
  background(0);
  noStroke();
}

function draw() {
  //background(0,5);
  fill(0);
  rect(0, 0, width/2, height);
  fill(0,25);
  rect(width/2, 0, width/2, height);

  fill(255);
  ellipse(random(width), random(height), 30, 30);
}

//declarations
var acolor;
var x,y;

function setup() {
  createCanvas(400, 400);

  //initializations
  acolor = color(200,0,150,11);
  y = 0;
  x = random(width);
}

function draw() {
  background(255,10);

  noStroke();
  fill(acolor);
  ellipse(x,y, 100, 100);
  y+=10;
  if(y > height){
    x = random(width);
    y = 0;
  }
}
