Lesson 19: Operators


These Operators can operate on the variables before and after the operator.

operators

+, -, *, \, +, -, %
The operator sum can add variables, numbers and strings. When you add a string and a number, it will become a string! 

                              // + sum
var a = b + c;                // sum variables
var d = 1 + 2;                // sum values
var e = f + 2;                // sum variable and value
var name= "name: " + "Janet"; // "name: Janet";
var nr  = "nr: "    + 1 ;     // "nr: 1"

                              // - subtract
var a = b - c;                // subtract variables
var d = 7 - 3;                // subtract values
var e = d - 2;                // subtract variable and value

                              // * product
var a = b * c;                // product of variables  (b x c)
var d = 7 - 3;                // product values
var g = d - 2;                // product variable and value

                              // / division
var a = b / c;                // a is result of b divide by c
var d = 9 / 2;                // d is result of 9 divided by 2, is 4.5
var g = d / 2;                // g is result of variable d divided by 2

                              // % modulus
var a = b % 2;                // returns the division remainder,
var d = 8 % 3;                // d is the remainder of 8 devided by 3 is 2
var g = d % 5;                // g is the remainder of d devided by 5 is 0,1,2,3 or 4.

If you devide an variable, the result will be a floating point value. The remainder of a modulus operation is allways an integer. 

//This is an example with the modulus operator.
function draw() {
  if(frameCount % 3 == 0){
    	fill(0,60);
    	noStroke();
  		ellipse(random(width), random(height), 20, 20);
  }
  else if(frameCount % 3 == 1){
    	fill(100,60);
    	stroke(100,0,0,100);
  		ellipse(random(width), random(height), 120, 120);
  }
  else if(frameCount % 3 == 2){
    	fill(200,60);
    	noStroke();
    	ellipse(random(width), random(height), 220, 220);
  }

assingment operators

=, +=, -=, *=, /=
The operator sum can add variables, numbers and strings. When you add a string and a number, it will become a string! 

var c = a + b;                // assign the result of a + b to c
c += 1;                       // add 1 to the value of c
var text = "start";
text += " and add more text." // result text is "start and add more text."
c -= 10;                      // subtract 10 from the value of c
c *= 4;                       // multiply the value in c 4 times
c /= 2;                       // devide the value of 4 by 2

Comparison operators

==, ===, !=, <=, =>
Comparison operators compare two variables or numbers, booleans or strings. the result of an logical operator is true or false.
True and False are booleans. (meaning: YES / NO or ON / OFF) true == 1; // is equal to 1 false == 0; // is equal to 0. 

a = b                  // is an assignment!
but
a == b;                // is true if the value of a is equal to the value of b;
d === e;               // is true if the value of d is equal to the value of e and d and e are of the same type.
f != g;                // is true if f is not equal to g;
h < i                  // is true if h is smaller then i;
j > k                  // is true if

Logical operators

&&, ||, !
With logical operators you can combine comparisons.
&& means AND
|| means OR 

if (a > 10 && a < 20 )      //is true if a is greater then 10 AND a is less then 20
if (b < 10 || b > 20 )      //is true if b is less then 10 OR b is greater then 20
