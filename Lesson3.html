Dots, Dots, Dots and Dots ...

You can create a dot with the point function. 

      point(x, y);

The point function has two params:

  • the x-position; how far in pixels from the left border?
  • the y-position; how far in pixels from the top?

the params (x,y) mark the center of the point. But before you can draw a dot you have to get your tools ready. You need a stroke function, so your drawingtool knows which color to use. You need a strokeWeight function, to tell your drawingtool how big your dot should be.

Remember: position (0,0) is in the top-left-corner. 

    function setup(){
      createCanvas(300,300);
    }

    function draw(){
      stroke(100);
      strokeWeight(130);
      point(75, 100);

      stroke(100);
      strokeWeight(300);
      point(100, 300);

      stroke(100);
      strokeWeight(130);
      point(225, 100);

      stroke(100);
      strokeWeight(300);
      point(200, 300);

    //three dots

      stroke(0);
      strokeWeight(20);
      point(100, 100);

      stroke(100);
      strokeWeight(20);
      point(150, 100);

      stroke(200);
      strokeWeight(20);
      point(200, 100);
    }

open sketch in openprocessing

The random function will pick a number between 0 and max, or between a minimum and a maximum. 

        random(255) // is a random number between 0 and 255.
        random(200,255) // is a random number between 200 and 255.

In the setup code of the example you can see I used a comment line. A comment line begins with // every thing behind the two slashes, the computer will ignore. It is important to comment your code. So when you read your code a month later you will understand what you were trying to do. Also whitespace will make your code easer to read.

You can open this example in openprocessing by clicking the link at the bottom of the page.
In openprocessing you can "FORK" a sketch, that is copy a sketch, change it and save it as your own.
