Lesson 5: Variables


Variables are used a lot in programming language, because they make your code flexible. x and y are two variables often used. They can represent every point in space, inside and outside your little canvas. 

        var x = -100;
        var y =  68000023;
A variable is like a 'box' where you can store a value.
You already know a couple of variables P5 is using, like windowWidth and windowHeight to store the value of the width and height of the window.
So do width and height, they store the size of your canvas. mouseX and mouseY to store the value of the mousepointer.

But now you will learn to create your own variables.

Variables

Before you can use a variable, you have to declare it: 

        var box;
        var x,y;
A variable can have any name, exept for the preserved words p5 and javascript are using.
And you have to initialize the variable: 
        box = 10;
        x = 0;
        y = 0;
If you create a variable within a function, the variable is only known within a this function, where you create your variable. 

  function setup(){
    createCanvas(300,300);
    background(100);
  }
  function draw(){
    //declaration (var y)
    //and initialization (y = 100).
    var y = 100;

    stroke(255);
    strokeWeight(2);
    line (0, y, width, y );
    y += 20;     //now y = 120
    line (0, y, width, y );
    y += 20;     //now y = 140
    line (0, y, width, y );
  }

open sketch in openprocessing

Now every time the draw function will draw the same lines, because in the beginning of the draw function y is initialized.
You can change this behaviour by declaring a variable outside the functions, before the setup function. Now we can initialize the variable y in the setup function before you will use it in the draw function.


open sketch in openprocessing 

  // declaration outside the functions
  var y;

  function setup(){
    createCanvas(300,300);
    background(100);
    // initialization before use
    // in the draw function.
    y = 100;
  }

  function draw(){
    stroke(255);
    strokeWeight(2);
    line (0, y, width, y );
    y += 20;  //add 20 to y
  }

  var x;          //declaration of the variable x;

Can you read this little program? Try to figure out what is happening, after you tried it. copy the code into an editor and see if you are right.

2. One_Line_Drawing

  • create a canvas as wide as your window and 400 pixels height.
  • First make a sketch on graph paper. It will save you a lot of time.
  • create a sketch in the same size, where 50 pixels is 1 cm.
  • Use a variable x for the begin points and end points of you lines.

    • line(x + 0, 300, x + 50, 300);
    • line(x + 50, 300, x + 150, 100);
    • etc.
  • After you have declared your x variable outside all the functions, you can start with x = -width in the setup function;
  • Now the line can move to the right, if we add a speed value to the x.

3. Line Patterns

  • Repeat lines and cross lines
  • Use a variable to add to the x values of the start and end point of the line.
  • Use a variable to add to the y values of the start and end point of the line.

4. Dancing Lines

  • Now you can slowly change a line.
  • Use variables and try it out.
  • Hint: You can make lines smaller then 1 with strokeWeight(0.5) or strokeWeight(0.1)
  • Hint: You can add values to variables smaller then 1, like 0.5 or 0.1
