Lesson 10: The mouse


Until now you have used p5 functions, like fill and ellipse or background, etc. You even changed two function of p5, the draw function and the setup function. You decided how they should work. p5 has an other function you can change. 

    function mousePressed(){
      // do something
    }

You need this function to let your code be interactive with a mouse.

  • You will learn

  • mousePressed
  • mouseIsPressed
  • mouseX, mouseY
  • pmouseX, pmouseY
  • boolean
  • true or false
  • not

You can create your own brushes. But you need to know a little trick. Your mousePosition is at mouseX, mouseY, but p5 also remembers the previous mousepointer position. It is stored in pmouseX, pmouseY. In this example the p5 variable mouseIsPressed is used. This is a boolean variable, this variable stores the value true or false depending if the mousebutton is pressed. If you hold your mousebutton down and drag your mouse over the screen, you can draw a continuous line. If you press and release your mousebutton the p5 function mousePressed is called and executed. Try this example and you will see the difference.

if

  function setup(){
    createCanvas(400, 400);

  }
  function draw() {
    if(mouseIsPressed){
      line(pmouseX, pmouseY, mouseX, mouseY);
    }
  }
  function mousePressed(){
    p.stroke(255);
    p.strokeWeight(5);
    p.noFill();
    p.ellipse(p.pmouseX, p.pmouseY, 60,60);
  }

Let's have a closer look at a boolean variabele. A boolean variabele stores the value true or false, thats the same as a result of a condition. 

  x === 3 is true or false.
  mouseIsPressed is true or false.

Internally in the computer true == 1 and false == 0; You can define your own booleans as variables. 

  var isReady = false;

  if(isReady){
    //stop
    noLoop();
  }
  //it is not good practice to write:
  if(isReady === true)  //but it is correct.

There is also a not, you can use by setting an exclamation mark in a condition. 

  if(!isReady)      //is true if isReady is false;
  if(x != 100)      //if x is not equal to 3

One more topic in this lesson. You can detect which way the mouse is going by comparing the current mouse position (mouseX, mouseY), with the previous mouse position (pmouseX, pmouseY).

    if(mouseX < pmouseX){  //less then
      //mouse is going to the left side.
    }
    if(mouseX > pmouseX){   //greater then
      //mouse is going to the right side.
    }
    if(mouseY > pmouseY){   //greater then
      //mouse is going down.
    }
    if(mouseY < pmouseY){   //less then
      //mouse is going up.
    }
